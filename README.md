# Translation-Different-ISH-Slice
By using the framework of CycleGAN and pix2pix to train the model with the large number of in situ hybridization(ISH) slices data of mouse brain in Allen Brain Altases, achieving the translation of different gene trend on one tissue ISH slice.  

* The repo of cyclegan: https://github.com/junyanz/pytorch-CycleGAN-and-pix2pix
* The link of allen brain altases: https://mouse.brain-map.org/
